## Non-Convex Neural Network (CIFAR-10)
The pre-trained parameters of ResNet-20 for CIFAR-10 are included in the folder. Please run each algorithm file separately. Use a fixed random seed to ensure the channel state remains the same.

Note: For training CIFAR-100, please use the aother ResNet-20 pre-trained parameters for CIFAR-100 provided in the folder.
